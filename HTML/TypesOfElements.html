TYPES OF ELEMENTS

BLOCK LEVEL

Block level elements start on a new line and inherits the size of their parent
Block elements have height and width
Width : parent
Height : content
Examples : heading, div , table, form etc

INLINE LEVEL

Inline level elements do not start on a new line and only take up as much height width as their content ex : a,b,i,p
